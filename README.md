# coincap-wrapper
Wrapper for coincap API. The code will run 2 servers: a json server (port 8080) for persisting data and a normal express server (port 80). once the server started, go to localhost/docs for api documentation.

## install
run 
`npm install`

## build
run 
`npm build`

## start
run 
`npm start`

## debug
run 
`npm start`
